Tunable Soft Equivariance with Guarantee

A lightweight mechanism that continuously interpolates between strict group equivariance and the original pretrained model, controlled by a single softness parameter. Implemented as forward-time weight filtering for modern vision backbones (ViT / ResNet / DINO / SegFormer) and scientific equivariant MLPs.

Read the paper → GitHub repo →

What is soft equivariance?

Many pretrained models break under group actions (e.g., rotations), while strict equivariance can be too restrictive. Soft equivariance enforces symmetry gradually by projecting weights to invariant/equivariant subspaces and interpolating back to the original weights with a single knob: softness ∈ [0, 1].

Single parameter
softness: 0.0 → equivariant, 1.0 → original
Forward-time filtering
No architecture change; wrap existing layers
Broad coverage
Vision + scientific groups (C4, SO(3), O(5), …)

How it works (schema)

Soft equivariance is applied at forward time by filtering (projecting) pretrained weights to invariant/equivariant subspaces, then interpolating between the projected and original weights. 

W_projected = P(W)
W_soft = softness · W + (1 - softness) · W_projected

softness = 0.0  → strict equivariance
softness = 1.0  → original pretrained weights

Code at a glance

The main implementation applies filters to pretrained layers at forward time. See the GitHub repository for training scripts, model wrappers, and group-specific constraint implementations. 

# Pseudocode sketch
model = load_pretrained_backbone(...)

# Wrap layers: Conv2d / Linear / Patch projection
model = wrap_with_filtered_layers(model, group="C4", softness=0.6)

# Run normally
y = model(x)
